num2aheui / string2aheui

개요

아희는 난해한 프로그래밍 언어 중 하나로서, 자세한 사항은 여기를 참조 바랍니다. num2aheui가 제공하는 기능은, 2~9를 이용하여 수를 만들기가 버거운 아희 개발자들을 위하여 숫자를 입력하면 해당 숫자를 스택에 만들어주는 아희 코드를 만들어주는 것입니다. string2aheui가 제공하는 기능은, 문자열을 입력하면 스택을 이용하여 해당 문자열을 print 해 주는 코드를 만들어주는 것입니다. 비록 프로그래밍 로직을 짜는 데 있어서 아무런 도움이 되지 않지만, 아희 개발자들에게 조금이나마 이 기능들이 도움이 되었으면 합니다.
